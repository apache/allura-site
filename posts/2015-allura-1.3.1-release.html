Apache Allura 1.3.1 released

Published: Mon 10 August 2015
Tagged: release

Apache Allura 1.3.1 has been released. It includes 3 security fixes and many small fixes and improvements. The major new features are:

  • CORS support, allowing broader access to the REST APIs.
  • A brand new Markdown editor with syntax highlighting and an editing toolbar.
  • New and updated docs for Installation and Administration
  • Ticket notifications include links to attachments.
